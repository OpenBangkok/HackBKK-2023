d# Objective
Traffy Fondue พร้อมเปิดรับทุกไอเดีย ต่อยอดสู่การให้บริการของกรุงเทพมหานคร เพื่อให้แพลตฟอร์มนี้เป็นศูนย์กลางให้บริการประชาชนในทุกมิติ ผ่านกิจกรรม Hackathon (แฮกกาธอน) ตลอด 3 วัน ในโครงการ HACK BKK 2023
